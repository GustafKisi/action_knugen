# Action Knugen
 

All hail Knugen

 ## Usage

 You can add the action to your Action YAML file like so: 
 ```
  - uses: GustafKisi/action_knugen@1.1
    with:
        message: '<your message />'
        token: ${{secrets.GITHUB_TOKEN}}
```

Both the `message` and the  `token` parameters are required.
